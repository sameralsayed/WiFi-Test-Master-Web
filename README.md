# 📶 WiFi Test Master Web

**A complete, sleek, fully functional web application** replicating the popular Android app “WiFi Test Master”.

## ✨ Features
- 📡 **Live Dashboard** – Current SSID, signal strength (dBm), fake real-time speeds  
- ⚡ **Speed Test** – Animated download/upload/ping test with progress bar  
- 🔍 **WiFi Analyzer** – Scan nearby networks with signal bars & security info  
- 📋 **Network Info** – IP, MAC, Gateway, DNS & more (realistic simulation)  
- 📶 **Signal Meter** – Beautiful animated signal bars  
- 📱 **100% Mobile-First** – Perfect on phones, tablets & desktop  
- 🌐 **SEO Optimized** – Full meta tags, keywords & fast loading  

## 🛠️ Technologies Used
- 🔨 **HTML5** – Semantic markup + responsive design  
- 🎨 **CSS3** – Dark neon theme with smooth animations  
- 📱 **Bootstrap 5** – Clean & fully responsive UI  
- ⚡ **JavaScript + jQuery** – Real-time tests & interactive scans  

## 🚀 How to Run
1. Download the 4 files into the **same folder**  
2. Open `index.html` in any modern browser  
3. Start testing your WiFi instantly!
